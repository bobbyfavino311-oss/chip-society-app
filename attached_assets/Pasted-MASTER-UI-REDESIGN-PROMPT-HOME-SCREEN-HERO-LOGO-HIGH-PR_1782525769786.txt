MASTER UI REDESIGN PROMPT — HOME SCREEN HERO LOGO (HIGH PRIORITY)

This is a major visual redesign of the Chip Society Home Screen and should be treated as a premium branding update. The objective is to replace the existing header logo with a much stronger hero element that immediately communicates the identity of the game.

The attached logo design is now the official direction for the Home screen.

Primary Goal

Completely replace the existing “Chip Society” logo at the top of the Home screen with the new handwritten neon logo shown in the reference image.

This logo becomes the centerpiece of the Home screen.

Everything beneath it should be repositioned accordingly.

Logo Recreation Requirements

Attempt to recreate the logo as accurately as possible using vector graphics instead of simply displaying a screenshot.

The AI should match:

• identical script style
• identical proportions
• identical line thickness
• identical underline beneath “Society”
• identical neon glow
• identical color palette

The logo should immediately feel like a premium casino resort sign.

Not a generic neon font.

Not cursive generated text.

Society

Not blurry.

Neon Motel Flash Animation

The logo should not remain static.

Instead it should behave like an authentic vintage motel sign.

Animation should be extremely subtle.

Examples:

• occasional tiny brightness flicker

• slight neon pulse

• one letter flickers for 50–100ms before stabilizing

• underline softly brightens

• glow intensity breathes

Never make it flashy.

Never blink the whole logo.

Never repeatedly flash.

It should feel like a real neon sign that’s been hanging outside a casino for years.

Think luxury—not arcade.

⸻

Animation Frequency

Very infrequent.

Every 10–20 seconds.

Randomized.

Never repetitive.

Do NOT approximate it poorly.

The supplied logo is preferable to a bad recreation.
