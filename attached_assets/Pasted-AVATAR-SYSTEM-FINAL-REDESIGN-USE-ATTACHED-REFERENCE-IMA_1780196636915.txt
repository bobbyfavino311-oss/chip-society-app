AVATAR SYSTEM FINAL REDESIGN
USE ATTACHED REFERENCE IMAGE

The attached image is the design direction.

DO NOT trace it.
DO NOT crop it directly.
DO NOT recreate it pixel-for-pixel.

Instead, create ORIGINAL artwork that matches the same visual philosophy, quality level, readability, and silhouette strength.

================================================================

CORE GOAL

Every avatar must be instantly recognizable at:

* profile size
* feed size
* gameplay table size
* leaderboard size
* notification size

A player should identify the symbol in less than one second.

If a symbol is confusing, abstract, or requires interpretation, redesign it.

================================================================

STYLE RULES

Create premium neon lounge symbols.

Think:

* luxury poker room
* Miami nightlife
* cocktail lounge
* Vegas after dark
* retro-futuristic neon
* collectible player badges

NOT:

* app icons
* logos
* clip art
* emojis
* abstract geometry
* decorative line art

================================================================

ICON DESIGN RULES

Every avatar must contain:

* one central subject only
* strong silhouette
* clean recognizable shape
* consistent line weight
* subtle neon glow
* circular composition
* dark background

REMOVE:

* random circles
* extra framing rings
* decorative overlays
* secondary symbols
* unnecessary geometric shapes
* clutter

The symbol itself must be the focus.

================================================================

VISUAL QUALITY

Generate all avatars as:

* vector quality
* razor sharp
* ultra high resolution
* anti-aliased edges
* crisp neon tubing
* no pixelation
* no blur
* no muddy colors

These assets will be cropped and used directly in the game.

Spacing between icons must be generous.

Leave enough empty space around every symbol so automated cropping never captures portions of neighboring avatars.
