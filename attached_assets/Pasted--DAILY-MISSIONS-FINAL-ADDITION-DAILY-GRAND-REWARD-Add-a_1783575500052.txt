# DAILY MISSIONS FINAL ADDITION — DAILY GRAND REWARD

Add a Daily Grand Reward system to the Daily Missions feature.

This reward is ONLY earned when the player successfully completes ALL FIVE Daily Missions within the same 24-hour Daily Mission cycle.

This is intended to be the highest-value daily reward and should encourage players to complete every mission before the daily reset.
