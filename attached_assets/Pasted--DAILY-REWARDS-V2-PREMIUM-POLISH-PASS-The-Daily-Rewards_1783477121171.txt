# DAILY REWARDS V2 — PREMIUM POLISH PASS

The Daily Rewards redesign is a major improvement, but it still feels slightly "UI generated." This update is a refinement pass only.

Do NOT redesign the layout.

Do NOT move the cards.

Do NOT change functionality.

Everything below is strictly visual refinement.

━━━━━━━━━━━━━━━━━━━━━━

# DESIGN GOAL

The Daily Rewards cards should feel like premium luxury reward cards.

The visual language should communicate:

• Depth
• Precision
• Premium materials
• Elegant restraint

The section should feel like it belongs inside a AAA mobile game.

Think:

Luxury watch.

Premium automotive dashboard.

Apple-quality material design.

Not flashy.

Not cartoonish.

━━━━━━━━━━━━━━━━━━━━━━

# THIN THE PROGRESS RINGS

The current progress rings are visually too heavy.

Reduce ring thickness by approximately 35–40%.

Increase the ring diameter slightly.

Reduce the opacity of the inactive portion.

Only the completed progress should be bright.

The ring should become a subtle accent rather than the main focal point.

━━━━━━━━━━━━━━━━━━━━━━

# CHANGE TO A 270° PROGRESS ARC

Replace the full circular progress ring with a premium 270° progress arc.

Leave the bottom portion of the circle open.

The arc should wrap around the top and sides only.

This immediately creates a cleaner, more premium look similar to a luxury watch complication.

The arc should fill smoothly as progress increases.

━━━━━━━━━━━━━━━━━━━━━━

# ENLARGE THE GLASS BADGE

Increase the glass badge behind the icon by approximately 10–15%.

This gives the custom icons more visual presence without increasing clutter.

The badge should remain centered inside the progress arc.

━━━━━━━━━━━━━━━━━━━━━━

# IMPROVE THE GLASS MATERIAL

The icon badge should feel like polished smoked glass.

Add:

• Soft top highlight
• Gentle inner shadow
• Slightly darker lower edge
• Very subtle cyan reflection
• Better glass depth

The badge should resemble a premium acrylic button.

━━━━━━━━━━━━━━━━━━━━━━

Daily Spin:

Streak:

The reward cards currently appear almost flat.

Introduce a very subtle vertical gradient.

Top:

Slightly lighter.

Slightly darker.

The card should feel more dimensional.
