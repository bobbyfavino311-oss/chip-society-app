# MASTER UI REDESIGN — SIMPLIFIED POKER NAVIGATION

The current navigation for poker variants requires too many unnecessary decisions.

Currently the flow is:

Variant
↓

AI Practice or Quick Play
↓

Quick Play opens another popup asking:

Who do you want to play against?

↓

AI or Live

↓

Stake Selection

This is confusing, repetitive, and adds unnecessary taps.

The backend already correctly supports AI and Live matchmaking.

This update is ONLY a navigation redesign.

Do NOT rewrite matchmaking or networking.

━━━━━━━━━━━━━━━━━━━━━━

# OBJECTIVE

Redesign the poker navigation to make starting a game feel immediate, modern, and intuitive.

Players should instantly understand whether they are playing:

• AI
or
• Live Players

without unnecessary popups.

━━━━━━━━━━━━━━━━━━━━━━

# REMOVE THE CURRENT LAYOUT

Delete the following from every poker variant:

• AI Practice
• Quick Play

Do not replace them with more buttons.

The current popup asking:

"Who do you want to play against?"

must also be removed completely.

Delete:

• AI Players popup option
• Live Players popup option
• Select Stakes popup button
• Cancel button

This entire screen should no longer exist.

━━━━━━━━━━━━━━━━━━━━━━

# NEW VARIANT LAYOUT

Each poker variant should have exactly the following structure:

Variant Icon

Variant Name

Short Description

AI / LIVE Mode Toggle

PLAY Button

Nothing else.

Example:

NO LIMIT HOLD'EM

52-card deck

Standard hand rankings

[ AI ]      [ LIVE ]

PLAY →

━━━━━━━━━━━━━━━━━━━━━━

# AI / LIVE TOGGLE

Replace the old buttons with a segmented control.

Example:

[ 🤖 AI ]    [ 🌐 LIVE ]

Only one option may be selected.

The selected option should glow using the variant's accent color.

This selector should remain visible at all times.

The player always knows what mode they're entering.

━━━━━━━━━━━━━━━━━━━━━━

# REMEMBER PLAYER CHOICE

The selected mode should persist.

If the player selects:

LIVE

Every poker variant should remember LIVE until the player changes it.

If they later switch to AI:

All poker variants now default to AI.

Do not ask every time.

Store this preference locally.

━━━━━━━━━━━━━━━━━━━━━━

# SINGLE PLAY BUTTON

Each poker variant should only have ONE Play button.

Example:

PLAY →

No additional prompts.

No extra confirmation.

━━━━━━━━━━━━━━━━━━━━━━

# PLAY BUTTON BEHAVIOR

If AI mode is selected:

PLAY

↓

Stake Selection

↓

Launch AI game immediately.

No matchmaking.

No popup.

━━━━━━━━━━━━━━━━━━━━━━

If LIVE mode is selected:

PLAY

↓

Stake Selection

↓

Matchmaking

↓

Join multiplayer table

↓

Fill remaining seats with AI if necessary.

No popup.

━━━━━━━━━━━━━━━━━━━━━━

# STAKE SELECTION

The stake selection screen remains exactly as it currently works.

The only change:

It opens directly after pressing PLAY.

Do not ask the player who they want to play against first.
