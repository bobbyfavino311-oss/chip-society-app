Theme Selection Screen Redesign

Completely redesign the Table Theme Selection screen. The current design feels too busy and outdated with the large icon boxes, thick borders, multiple color dots, oversized buttons, and long descriptions. I want this screen to feel like a premium game customization menu rather than a settings page.

Do not redesign the overall navigation or functionality—only redesign the presentation of the theme cards.

⸻

Overall Design Direction

Design inspiration:

* Apple Wallet
* Apple TV
* PlayStation Store
* Call of Duty Loadouts
* Clash Royale cosmetics
* Modern AAA game skin selectors

The experience should feel:

* Premium
* Minimal
* Elegant
* Easy to browse
* Luxury casino aesthetic

There should be much more whitespace and less visual clutter.

⸻

Remove the Large Icon Box

Completely remove the square icon box on the left.

Instead, every theme card should begin with a wide panoramic preview banner (approximately 3:1 aspect ratio) that displays a miniature preview of the actual poker table.

This preview should be generated from the real gameplay assets used by that theme.

Do not use generic icons anymore.

⸻

Live Table Preview

Each preview should display a simplified version of the actual table including:

* Table felt/background
* Community card area
* Signature community card border
* Table trim
* Theme colors
* Background artwork

Do NOT display:

* Player cards
* Chips
* Buttons
* UI
* Text overlays

The preview should immediately tell the player exactly what the table looks like in-game.

No oversized borders.

⸻

Animations

When scrolling:

Button transitions from “Equip” to “✓ Equipped.”

Very smooth.

⸻

Future Scalability

Design this layout so it scales cleanly to dozens or even hundreds of themes without becoming cluttered.

The new design should emphasize the actual table artwork instead of icons, making it easy for players to browse, compare, and collect themes at a glance.

The final result should feel closer to browsing premium cosmetic skins in a modern AAA game than navigating a traditional settings menu.
