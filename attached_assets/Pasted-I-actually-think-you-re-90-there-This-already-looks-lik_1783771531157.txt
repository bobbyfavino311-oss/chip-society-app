HOME SCREEN POLISH PASS — LAYOUT REFINEMENT (DO NOT REDESIGN)

The Home screen visual style is now very close to the desired look.

Do NOT redesign the page.

This is strictly a layout polish pass to improve spacing, proportions, and visual hierarchy.

⸻

1. TRENDING POSTS ARE BEING CLIPPED

This is currently the biggest issue.

The bottom of the Trending post cards is being cut off because there isn’t enough vertical space allocated.

Fix

Increase the vertical height allocated to the Trending carousel by approximately 20–25%.

The full post card—including:

* avatar
* username
* badge
* body text
* likes
* timestamp
* rounded bottom corners
* bottom shadow

must always remain completely visible.

Nothing should ever be clipped.

The shadow beneath the card should also remain visible.

The carousel should feel like it floats above the page instead of being cropped.

⸻

2. DAILY STREAK CARD IS TOO TALL

The Daily Spin and Daily Streak cards currently dominate too much vertical space.

They should feel like dashboard widgets, not feature cards.

Reduce the height of BOTH reward cards by approximately 18–22%.

Keep the width exactly the same.

Only reduce height.

⸻

3. REBALANCE THE CONTENT INSIDE THE REWARD CARDS

After shortening the cards:

Slightly reduce

• ring diameter (~8%)

• icon size (~5%)

• spacing above the title

Everything should feel tighter.

⸻

There is currently too much empty space underneath it.

Final Design Goal

The Home screen should feel like a carefully balanced premium dashboard where every section flows naturally into the next. No content should ever be clipped, oversized, or cramped. The Daily Rewards widgets should read as compact status cards, while the Trending feed remains the visual focal point. The result should look intentionally composed, with consistent spacing, perfect proportions, and the same luxury glass aesthetic throughout.
