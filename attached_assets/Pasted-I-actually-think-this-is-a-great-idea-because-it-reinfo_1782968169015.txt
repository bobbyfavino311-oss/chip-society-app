I actually think this is a great idea because it reinforces that Chip Society isn’t just Texas Hold’em. As you add more poker variants, your tournaments should showcase them. Players who prefer Omaha or Joker Hold’em should have tournaments built specifically for those formats rather than forcing everyone into standard Hold’em.

Here’s how I’d expand the tournament system:

⸻

NEW TOURNAMENT VARIANTS

Expand the tournament system to support multiple poker variants.

Each tournament card should clearly display the game variant being played.

Add a small game badge beneath the tournament title.

Examples:

♠ Traditional Hold’em

🃏 Joker Hold’em

🂠 Omaha Hold’em

This should be immediately visible before a player joins.

MATCHMAKING

Never mix poker variants.

An Omaha tournament only seats Omaha players.

A Joker tournament only seats Joker Hold’em players.

Traditional Hold’em tournaments only seat Traditional Hold’em players.

Each tournament should launch using the correct game engine, rules, hand evaluator, AI behavior, achievements, and statistics for that specific variant.

DESIGN GOAL

Chip Society tournaments should showcase the full range of poker variants rather than centering exclusively on Texas Hold’em. Players should instantly recognize which variant a tournament uses, and each event should play according to that variant’s official rules, creating distinct tournament experiences for Traditional Hold’em, Omaha Hold’em, and Joker Hold’em.
