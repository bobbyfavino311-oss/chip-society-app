CHIP SOCIETY — LIGHT MODE / DARK MODE THEME SYSTEM FIX
GLOBAL READABILITY PATCH
==============================================================

The current light mode version of the app is broken visually.

Major issue:
Some text and UI elements become unreadable because dark-mode colors are being used on light backgrounds.

This needs to be fixed globally across the entire app.

IMPORTANT:
This theme fix should apply to every screen EXCEPT live table gameplay.

Live table gameplay should keep its current dark casino table style.

==============================================================
1. CURRENT LIGHT MODE PROBLEMS
==============================================================

The light mode currently has:

- white text on pale backgrounds
- low contrast labels
- invisible section headers
- unreadable card text
- transparent gray overlays that wash out text
- bottom navigation readability problems
- feed category tabs with weak contrast
- profile stats with poor color balance
- tournament cards that clash with the background
- daily spin / streak cards that fade into the page

This makes the app feel unfinished and hard to use.

==============================================================
2. REQUIRED THEME SYSTEM
==============================================================

Build a proper theme system with two modes:

DARK MODE:
- current neon casino dark theme

LIGHT MODE:
- inverted clean casino light theme

The theme system must use shared design tokens instead of hardcoded colors.

Create global theme variables for:

- background
- surface
- cardBackground
- textPrimary
- textSecondary
- textMuted
- border
- accentCyan
- accentPink
- accentPurple
- success
- warning
- danger
- navBackground
- tabActive
- tabInactive

==============================================================
3. LIGHT MODE DESIGN DIRECTION
==============================================================

Light mode should NOT simply turn the background white.

It should feel like:
- premium frosted casino glass
- soft lavender luxury UI
- clean neon mobile interface
- readable modern social app

Use:

BACKGROUND:
soft lavender / off-white

CARDS:
deep purple or frosted white surfaces

TEXT:
dark navy / deep purple for primary text

ACCENTS:
cyan, pink, purple, gold

BORDERS:
soft purple / cyan outlines

==============================================================
4. READABILITY RULES
==============================================================

Every screen must pass readability rules:

- dark text on light backgrounds
- light text on dark cards
- no washed-out transparency
- no white text on pale lavender
- no gray text below readable contrast
- no icons disappearing into background

All UI must remain readable at normal phone brightness.
