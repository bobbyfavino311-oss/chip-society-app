CHIP SOCIETY — CHARACTER AVATAR SYSTEM FINAL OVERHAUL
CRITICAL FIXES + PROFILE IMAGE SYSTEM
==============================================================

The current avatar system is improved, but it is still NOT production ready.

This update is intended to finalize the character system and prepare it for launch quality.

==============================================================
SECTION 1 — REMOVE BROKEN CROPPING
==============================================================

Current Problems:

- Character numbers are visible inside portrait artwork.
- Character names are partially embedded into images.
- Several portraits are cropped incorrectly.
- Some portraits show parts of neighboring character cards.
- The last Legendary characters are visibly sliced from a larger image sheet.
- Portrait framing is inconsistent.

This makes the entire system feel unfinished.

==============================================================
FIX
==============================================================

STOP slicing portraits from a master image sheet.

Every character portrait must be generated and exported as an INDIVIDUAL asset.

Each portrait should be:

1080 x 1440
3:4 ratio
Portrait orientation

Each image should contain ONLY:

- Character portrait
- Background
- Lighting
- Clothing
- Character personality

Do NOT include:

- Numbers
- Character names
- XP requirements
- Lock icons
- UI overlays
- Rarity badges

All of those elements must be rendered separately by the UI layer.

==============================================================
SECTION 2 — PORTRAIT CROPPING RULES
==============================================================

Every portrait must follow identical framing.

Required Composition:

- Full head visible
- Hair visible
- Shoulders visible
- Upper chest visible
- Face centered
- Eyes positioned in upper third

Do NOT allow:

- forehead clipping
- chin clipping
- cut-off hats
- cut-off shoulders
- half faces
- neighboring portraits visible

Character should occupy:
60–70% of frame.

The portrait should still be recognizable when scaled down.
