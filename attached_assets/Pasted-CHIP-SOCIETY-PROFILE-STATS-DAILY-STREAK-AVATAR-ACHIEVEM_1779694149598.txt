CHIP SOCIETY — PROFILE STATS + DAILY STREAK + AVATAR + ACHIEVEMENT PROGRESSION PATCH
==============================================================

Update the player profile, daily streak system, avatar selection, and achievement tracking system.

This patch must fix progression accuracy and make achievements harder to earn so players have stronger incentive to keep playing.

- Biggest Tournament Prize Won

Tournament Record:

Win Rate:
60%

Best Finish:
1st Place

==============================================================
2. DAILY STREAK COUNTDOWN TIMER
==============================================================

Add a 24-hour countdown timer to the daily streak reward feature.

After a player claims the daily streak prize:

- lock the next reward
- start a visible 24-hour countdown
- show exact time remaining

Example:

Next Daily Reward In:
23h 14m 08s

When the countdown reaches zero:

- enable the next reward
- show “CLAIM” button
- trigger subtle notification badge

The countdown must persist after closing and reopening the app.

Use server time if available.
If no backend exists yet, use local saved timestamp for beta.

4. ACHIEVEMENT PROGRESSION OVERHAUL
==============================================================

The current achievements unlock too easily.

All hand-related achievements should now require multiple completions before unlocking.

Instead of unlocking after one time, each achievement should track progress.

Example:

Pocket Rockets
Win a hand with One Pair.
Progress: 1 / 10

Double Down
Win a hand with Two Pair.
Progress: 2 / 10

Set the Table
Win a hand with Three of a Kind.
Progress: 4 / 10

==============================================================
5. HAND-RELATED ACHIEVEMENTS REQUIRE 10 WINS
==============================================================

All hand-related achievements should require 10 successful completions.

Apply 10-count requirement to:

- One Pair
- Two Pair
- Three of a Kind
- Straight
- Flush
- Full House
- Four of a Kind
- Straight Flush

Each must show progress:

0 / 10
1 / 10
2 / 10
...
10 / 10

Achievement unlocks only at 10 / 10.

==============================================================
6. ROYAL FLUSH EXCEPTION
==============================================================

Royal Flush achievement should only require 1 completion.

Royal Flush is rare enough and should remain a single-completion legendary achievement.

Display:

Neon Royalty
Win a hand with a Royal Flush.
Progress: 0 / 1

When achieved once:

Progress: 1 / 1
Achievement unlocked.

8. CLAIM BUTTON REQUIREMENT
==============================================================

Achievements should NOT automatically pay rewards.

When progress reaches required amount:

- mark achievement as complete
- show “CLAIM” button
- player taps CLAIM
- reward is added to balance
- XP is added to profile
- achievement count updates correctly

This prevents confusion and improves reward pacing.

==============================================================
9. ACHIEVEMENT TRACKING ACCURACY
==============================================================

Achievements must only progress when the player actually earns the matching hand result.

Example:

If player wins with Two Pair:
Only Double Down should increase.

Do NOT increase:
- One Pair
- Straight
- Flush
- Full House

The system must evaluate the final winning hand correctly.

==============================================================
10. ACHIEVEMENT COUNT MUST UPDATE PROFILE
==============================================================

Player profile achievement count must reflect actual completed achievements.

Rules:

- incomplete achievements do NOT count
- claimable achievements may count only after unlocked
- claimed achievements remain counted
- no duplicate counting
- no false achievements

Example:

If player completed 8 achievements:
Profile must show exactly 8.

server-side persistence is preferred.
