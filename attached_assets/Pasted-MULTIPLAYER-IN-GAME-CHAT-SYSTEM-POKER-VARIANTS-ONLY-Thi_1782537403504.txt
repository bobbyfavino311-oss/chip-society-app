MULTIPLAYER IN-GAME CHAT SYSTEM (POKER VARIANTS ONLY)

This feature applies only to multiplayer poker variants.

Do NOT add this feature to any Casino Games.

Supported games include:

* Traditional Hold’em
* Short Deck Hold’em
* Omaha Hold’em
* Joker Hold’em
* Future multiplayer poker variants

⸻

Goal

Add a lightweight in-game chat system that allows players to communicate during hands without becoming distracting or toxic.

The system should feel polished, modern, and easy to use.

⸻

Chat Icon

Add a small Chat Bubble icon in an unused area of the gameplay screen.

Requirements:

• Small and unobtrusive
• Matches Chip Society neon UI
• Cyan outline
• Slight glow
• Circular button
• Never blocks cards or betting controls

Recommended placement:

Bottom-right corner above the betting controls.

⸻

Opening Chat

Tapping the chat icon slides up a compact chat panel from the bottom.

The panel should cover only the lower portion of the screen so gameplay remains visible.

Smooth animation.

Approximately 250–300ms slide animation.

⸻

Chat Layout

The chat panel contains:

Top

• Recent messages

Bottom

• Quick Chat buttons

• Text input field

• Send button

⸻

Quick Chat Messages

Players can instantly send one-tap preset messages.

No typing required.

Preset messages:

• Nice hand!
• Good luck!
• Well played.
• Nice bluff.
• I should’ve folded.
• Tough river.
• All in!
• GG!

Messages send immediately with one tap.

⸻

Custom Messages

Players may also type their own messages.

Maximum length:

50 characters

No multiline messages.

Single line only.

Send button becomes active once text exists.

Overall Goal

The feature should make multiplayer tables feel alive and social without overwhelming gameplay. Players should be able to congratulate good hands, bluff, joke, or interact briefly while remaining focused on the action. Chat should feel fast, clean, and premium, reinforcing Chip Society as a true Social Poker Network rather than just another poker app.
