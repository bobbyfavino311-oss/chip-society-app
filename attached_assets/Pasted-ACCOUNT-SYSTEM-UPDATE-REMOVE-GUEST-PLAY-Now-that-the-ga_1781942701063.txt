ACCOUNT SYSTEM UPDATE — REMOVE GUEST PLAY

Now that the game has a server, remove the Play as Guest option completely.

All players must create an official Chip Society account before playing.

The goal is simple:

No email.
No password.
No complicated signup.
Just username + 4-digit PIN.

━━━━━━━━━━━━━━━━━━━━━━

REMOVE GUEST MODE

Remove:

• Play as Guest
• Guest Account
• Guest Login
• Guest Starting Chips
• Guest Profile
• Guest Restrictions
• Guest Mode references anywhere in the app

Players should no longer be allowed to enter the game without creating an account.

━━━━━━━━━━━━━━━━━━━━━━

Every player must create an account using:

That is the entire signup flow.

Do NOT require:

USERNAME RULES

Username must be:

• 3–20 characters
• Letters allowed
• Numbers allowed
• Underscores allowed
• Unique across the server

Do not allow:

• Duplicate usernames
• Offensive names
• Empty usernames
• Special symbols

━━━━━━━━━━━━━━━━━━━━━━

PIN RULES

PIN must be:

• Exactly 4 digits
• Numbers only

Examples:

2486
1047
9001

Do not allow:

• Letters
• Symbols
• Fewer than 4 digits
• More than 4 digits

━━━━━━━━━━━━━━━━━━━━━━

PIN SECURITY

Do not store the PIN as plain text.

Store it securely as a hashed value on the server.

The real PIN should never be visible in the database.
